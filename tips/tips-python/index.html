tqdm

tqdm 是一个用于在 Python 中添加进度条的库。它的名字是 “taqaddum” 的缩写，意为 “进步”。tqdm 的主要作用是在循环迭代等长时间运行的任务时，提供一个可视化的进度条，让用户能够清晰地看到任务的进度。
